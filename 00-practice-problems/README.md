# Practice Problems

A collection of mixed problems for daily practice and warm-up.

## Purpose
- Daily coding practice
- Interview preparation
- Mixed difficulty problems
- Quick problem-solving

## Categories
- Easy (Beginner-friendly)
- Medium (Intermediate)
- Hard (Advanced)

## Naming Convention
Use format: `difficulty-problem-name.js`

Examples:
- `easy-fizzbuzz.js`
- `medium-longest-substring.js`
- `hard-trapping-rainwater.js`
